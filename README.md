
# 📚 Book Suggestion App

Welcome to the **Book Suggestion App**, a Python-based project that recommends books to users based on their preferred genre, minimum rating, and publication year. This app fetches real-time data using the Google Books API, processes it using Pandas, and suggests a random book based on user preferences.

---

## 🧠 Project Objective

The goal of this project is to apply Python concepts such as API integration, data processing with Pandas, user interaction, error handling, and clean code practices to build a functional and interactive book recommendation system.

---

## 🚀 Features

- 🔍 **Genre-Based Book Search**: Users can search for books in a specific genre (e.g., fiction, science, history).
- ⭐ **Rating Filter**: Set a minimum average rating (1 to 5) to filter out low-rated books.
- 📅 **Year Filter**: Filter books by their publication year.
- 🎲 **Random Suggestion**: Get a random book suggestion from the filtered list.
- ⚠️ **Error Handling**: Robust handling of network errors and invalid inputs.
- 📊 **Data Processing**: Uses Pandas to organize, clean, and filter data efficiently.

---

## 🧰 Tools & Technologies Used

- **Python 3**
- **Requests** – for calling the Google Books API
- **Pandas** – for data cleaning and filtering
- **Google Books API** – to fetch book data (title, authors, ratings, publication year)
