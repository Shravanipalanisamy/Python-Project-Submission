# 📘 Python Capstone Project: Book Suggestion App

This is the final project for my Python certification course.

## ✅ Project Overview

The **Book Suggestion App** fetches book data using the Google Books API and suggests a random book based on user preferences like genre, rating, and publication year.

##  Features

- API integration using Google Books
- Data handling using `pandas`
- User input for filtering books
- Randomized book suggestion
- Error handling for invalid inputs and API issues

##  Requirements

- Python 3.x
- `requests`
- `pandas`

pip install requests pandas
